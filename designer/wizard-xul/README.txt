This is a milestone build of the Pentaho Report Designer. The Pentaho Report Designer
simplifies and streamlines the report creation process, making it easy for report
authors to quickly create sophisticated, visually rich reports based upon
JFreeReports. It was originally developed by gridvision engineering Gmbh and is
now developed by Pentaho Corporation provided under the terms of the Mozilla
Public License, Version 1.1.

As a snapshot of the progress towards the next official version of the Pentaho Report 
Designer, this build is intended to present the new features, improvements, and fixes 
to the community for review and testing. A production quality build of the current 
Pentaho Report Designer 1.2 can be found at http://www.pentaho.org/download/.

This list represents the major changes and is not meant to be all inclusive:
